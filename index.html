Capabilities

What it tracks

Collection Stats

WebSocket listener for real-time listings, offers, sales, and transfers on any OpenSea-indexed collection.

📁
Data Export
Snapshot collections to CSV or local SQLite — useful for charting, backtesting, or offline research.

Usage

Start querying in minutes

src/client.py 
# Fetch collection stats from OpenSea V2
import httpx
from os import environ

class OpenSeaClient:
    base = "https://api.opensea.io/api/v2"

    def __init__(self):
        self.headers = {
            "x-api-key": environ.get("OPENSEA_API_KEY", ""),
            "accept": "application/json"
        }

    def get_collection_stats(self, slug: str):
        res = httpx.get(
            f"{self.base}/collections/{slug}/stats",
            headers=self.headers
        )
        res.raise_for_status()
        return res.json()

# Usage
client = OpenSeaClient()
stats = client.get_collection_stats("pudgy-penguins")
print(stats["total"]["floor_price"])  # → 11.2
Method Endpoint Description
GET /collections/{slug}/stats Floor price, volume, owners, supply
GET /events/collection/{slug} Sales and listing history
GET /nfts/{chain}/{address}/{id} Token-level metadata
GET /collections/{slug} Collection details and contract info
WS wss://stream.openseabeta.com/socket Real-time events feed
